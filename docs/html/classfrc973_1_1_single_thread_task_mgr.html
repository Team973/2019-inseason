frc973::SingleThreadTaskMgr::SingleThreadTaskMgr ( RobotState &  stateProvider,
double  loopPeriod = DEFAULT_PERIOD,
bool  warnSlow = false 
)

Initialize a Single Threaded Task Manager. The SingleThreadTaskMgr creates its own pthread in which to run all of its registered CoopTask's. Registered CoopTask's are guaranteed to run sequentially (i.e., no two tasks will be run at the same time). Registered CoopTask's will not start running until you call SingleThreadTaskMgr.Start.

Parameters
stateProviderThe current RobotState.
loopPeriodThe interval (in seconds) at which to schedule each period.
warnSlowDetermines whether to issue a warning when the tasks run longer than their alloted time.
